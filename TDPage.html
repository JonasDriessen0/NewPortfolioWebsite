Generic Tower Defense

A simple tower defense game

Introduction to the project

Generic Tower Defense is a project which I made for a school assignment to create a tower defense game. GenericTD focuses on having a clean and simple art style while still being fun and enjoyable to play. I learned a lot during the development of this project about not only how to make better code but also how to use Git more efficiently.

Gameplay

The gameplay consists of you having to stop ever-increasing waves of enemies from getting to your "Base". You can defend against the incoming enemies by buying and placing turrets along the path to your base.

Buying Turrets

You can buy turrets with a currency you get from killing enemies, which is called "Metal".

There are currently 2 different turrets you can buy.

Turret Placing

When you buy a turret, you go into placing mode, where a place icon will be following your mouse location along a grid, and when you click, you put down the turret on the grid where you clicked.

Set the variable "mousePosition" as the position of the mouse.

Move the place indicator along the grid smoothly with the help of the DotTween extension.

Place the turret when you click the left mouse button.

Outcome

While developing this project, I learned a lot of new skills in and outside of Unity, and I am very pleased with the end result.
